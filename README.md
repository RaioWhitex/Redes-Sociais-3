# 🌐 Neumorphism Social Media Icons UI

Um projeto moderno de **Ícones de Redes Sociais com Neumorphism Interativo**, desenvolvido para praticar conceitos de **HTML5** e **CSS3**, explorando efeitos de profundidade, sombras suaves e estados visuais de interação.

---

## 📸 Preview do Projeto

Interface contendo:

* Ícones de redes sociais em estilo Neumorphism
* Efeito de destaque para item ativo
* Borda colorida com foco visual
* Sombras suaves e realistas
* Layout horizontal centralizado
* Design clean e moderno

---

## ✨ Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **Font Awesome**

---

## 🎨 Características do Design

* Estilo Neumorphism (Soft UI)
* Efeito de profundidade através de sombras duplas
* Destaque visual para ícone selecionado
* Animações suaves de transição
* Interface minimalista e elegante
* Paleta de cores neutra com detalhes vibrantes

---

## 🚀 Objetivo do Projeto

Praticar habilidades de desenvolvimento front-end através da criação de componentes modernos e interativos, aplicando técnicas de Neumorphism, efeitos de hover e estados ativos utilizando apenas HTML e CSS.

---

## 📚 Funcionalidades

* Exibição de ícones sociais
* Destaque visual para item ativo
* Efeito de foco com borda colorida
* Sombras externas suaves
* Estrutura reutilizável para portfólios e landing pages
* Compatibilidade com diferentes tamanhos de tela
* Código simples e de fácil personalização

---

## 🎯 Conceitos Aplicados

* Neumorphism (Soft UI)
* Flexbox
* Pseudo-classes CSS
* Box Shadow
* Transições CSS
* Design Responsivo
